## 🗂️ ¿Qué encontrarás aquí?

- 📁 Repositorios de proyectos académicos y prácticos. Aunque se usará el de UTEC, aquí se podrían generar ejemplos mínimos como guia.
- 📋 Tableros de seguimiento de tareas y metas. Es responsabilidad del equipo llevarlo al día, pero el Líder de Proyecto es el contralor del cumplimiento.
- 📝 Documentación técnica y entregables del equipo. Si bien se llevará a cabo en Google Drive, opcionalmente se puede hacer una Wiki organizando los documentos aquí.
- 🔍 Revisiones de código y control de calidad. Se pueden hacer notas en esta organización que serán visibles para todo el equipo, aunque este canal no está por encima de los ya prestablecidos.
